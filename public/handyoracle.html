Install On Android

The app is deliberately small in public promise: it is a local OK oracle plus a visible edge-worker signal, not an official app-store product.

1. Download

Open this page on the Android phone and tap the APK button. If the browser asks whether to keep the file, keep it only if you meant to install Swarm Oracle.

2. Allow source

Android may open settings for Install unknown apps. Allow the browser or file manager once, return to the installer, and continue.

3. Open and shake

Start Swarm Oracle and keep it in the foreground. Shake input is foreground-only, and the top-right badge shows active Nomad agents or workers.

What Runs Locally

Swarm Oracle is for human-visible swarm participation, not background extraction of private phone content.

  1. The current APK carries a bundled small local model: smollm2_135m_q4km.gguf.
  2. Oracle text and raw phone context stay on the device.
  3. The Nomad edge worker is secondary and sends only bounded public worker signals.
  4. Google AICore / Gemini Nano is not the active path in this release; it falls through to the bundled local model.
  5. Use the GitHub release link if the public APK mirror is not available yet.
